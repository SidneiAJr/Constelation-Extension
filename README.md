# 🚀 Constellation Project Creator | Backend Only

**Cria estruturas de projeto backend completas e prontas para uso direto do VS Code — sem precisar criar pastas manualmente.**

---

## ✨ Funcionalidades

- 🎯 **6 frameworks**: JavaScript (Node.js/Express), TypeScript (Node.js/Express), Java (Spark Framework), Java (Spring Boot), PHP (Slim Framework), C# (ASP.NET Core)
- 🏗️ **4 arquiteturas**: MVC, DDD, Clean Architecture, Hexagonal
- 📁 **Estrutura completa**: Controllers, Services, Repositories, Models, Middlewares, Routes, Config, Utils, Schemas, Types, Exceptions
- 🧩 **MVC Universal**: Repository → Service → Controller gerados automaticamente para todos os frameworks com métodos padrão (create, findById, findByEmail, findByUsername, findAll, update, delete)
- 🔐 **Auth pronto**: JWT utils, middleware de autenticação e validação com Zod (TS/JS)
- 📄 **Arquivos base**: README.md, .gitignore, .env, .env.example, Dockerfile
- ⚡ **Um comando**: Escolha o nome, framework, arquitetura e pronto — código na hora

---

## 📦 Como Usar

1. Pressione `Ctrl+Shift+P` (ou `Cmd+Shift+P` no Mac)
2. Digite `Constellation: Criar Novo Projeto Backend`
3. Digite o nome do projeto
4. Escolha o framework
5. Escolha a arquitetura
6. Pronto — sua estrutura está criada e pronta para desenvolvimento

---

## 🛠️ Comandos

| Comando | Descrição |

| `Constellation: Criar Novo Projeto Backend` | Cria um novo projeto backend com estrutura completa |

## 📚 Stack gerada por framework

| Framework | Principais dependências |
|-----------|------------------------|
| TypeScript | Express, TypeORM, MySQL2, JWT, Bcrypt, Winston, Axios, Zod |
| JavaScript | Express, MySQL2, JWT, Bcrypt, Axios, Zod |
| Java Spark | Spark Framework, Gson, JWT, BCrypt, MySQL |
| Java Spring | Spring Boot, Spring Security, JPA, MySQL, JWT, Lombok |
| PHP Slim | Slim 4, Firebase JWT, PHP dotenv |
| C# ASP.NET | ASP.NET Core 8, EF Core, JWT Bearer, BCrypt |
