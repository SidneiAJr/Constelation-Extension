# 🚀 Constellation Project Creator | Backend Only

**Cria estruturas de projeto backend completas e prontas para uso direto do VS Code — sem precisar criar pastas manualmente.**

---

## ✨ Funcionalidades

- 🎯 **6 frameworks**: JavaScript (Node.js/Express), TypeScript (Node.js/Express), Java (Spark Framework), Java (Spring Boot), PHP (Slim Framework), C# (ASP.NET Core)
- 🏗️ **4 arquiteturas**: MVC, DDD, Clean Architecture, Hexagonal
- 📁 **Estrutura completa**: Controllers, Services, Repositories, Models, Middlewares, Routes, Config, Utils
- 📄 **Arquivos base**: README.md, .gitignore, .env.example, Dockerfile (opcional)
- ⚡ **Um comando**: Escolha o nome, framework, arquitetura e pronto — código na hora

---

## 📦 Como Usar

1. Pressione `Ctrl+Shift+P` (ou `Cmd+Shift+P` no Mac)
2. Digite `Constellation: Criar Novo Projeto Backend`
3. Digite o nome do projeto
4. Escolha o framework
5. Escolha a arquitetura
6. Pronto — sua estrutura está criada

---

## 🛠️ Comandos

| Comando | Descrição |
|---------|-----------|
| `Constellation: Criar Novo Projeto Backend` | Cria um novo projeto backend com estrutura completa |

| Requisito | Versão |
