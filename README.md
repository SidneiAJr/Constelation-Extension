# 🚀 Constellation Project Creator

**Creates complete, ready-to-use project structures directly from VS Code — no more manual folder setup.**

---

## ✨ Features

- 🎯 **5 frameworks**: Node.js (Express), Java Spring Boot, PHP (Slim), C# ASP.NET Core, TypeScript
- 📁 **Complete folder structure**: Controllers, Models, Services, Repositories, Middleware, Config, Utils, DTOs, Validators, Routes
- 🎨 **Frontend skeleton included**: HTML, CSS, JS — empty and waiting for your code
- 📄 **Base files**: README.md, .gitignore, .env.example, LICENSE
- ⚡ **One command**: Pick a name, pick a framework, start coding

---

## 📦 How to Use

1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
2. Type `Constellation: Create New Project`
3. Enter your project name
4. Select your framework
5. Done — your structure is ready

---

## 🛠️ Commands

| Command | Description |
|---------|-------------|
| `Constellation: Create New Project` | Creates a new project with complete folder structure |

---

## 🔧 Requirements

| Requirement | Version |
|-------------|---------|
| VS Code | 1.85.0+ |
| Node.js | Any (for JS/TS projects) |
| Java | 17+ (for Spring Boot) |
| PHP | 8+ (for Slim) |
| .NET | 8+ (for C#) |
