# Change Log

All notable changes to the "constellation-project-creator" extension will be documented in this file.

## [0.0.2] 

### Added
- 📁 More complete project structure
- 🏗️ Support for multiple architectures (preparation)
- 📝 Better documentation in README

### Changed
- 🔧 Downgrade VS Code engine support to v1.85.0
- 📦 Optimized folder structure for better organization

### Fixed
- 🐛 Compatibility issues with older VS Code versions
- 🔨 Fixed path resolution on Windows

### Added
- 🎉 Initial release
- 🏗️ MVC architecture support
- 📁 Basic project structure
- 🚀 Support for JavaScript and TypeScript
- ☕ Java Spring Boot support
- 🐘 PHP Slim support
- 🟦 C# ASP.NET Core support

### Features
- ✅ Automatic folder creation
- ✅ Pre-configured package.json

- ✅ Environment variables setup
- ✅ Git ignore pre-configured
